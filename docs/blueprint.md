# **App Name**: UniTool Launchpad

## Core Features:

- Feature Showcase Grid: An interactive and responsive grid showcasing 40+ utility cards including WhatsApp Status Saver and Poki Games with polished UI elements.
- AI Smart Discovery Tool: A natural language search interface where the tool reasons to intelligently recommend specific utilities or document tools based on user intent.
- App Store Download Center: A centralized hub featuring optimized CTA buttons for downloading the UniTool APK and direct links to the Google Play Store.
- AdMob Compliance Portal: A dedicated endpoint for hosting the app-ads.txt file required for official AdMob crawler verification and monetization.
- Legal and Privacy Hub: Specialized pages for Privacy Policy and Terms of Service that dynamically reference compliance content from the developer's GitHub repository.
- Privacy Architecture Showcase: An educational UI segment explaining the on-device AES encryption and the privacy-first architecture of the Private Media Vault.

## Style Guidelines:

- Primary Color: Vibrant Electric Violet (#8E7CFF) to evoke high-tech sophistication and versatility.
- Background Color: Deep Space Night (#0E0F14), a professional dark-mode canvas using desaturated indigo.
- Accent Color: Luminous Azure (#4FACFE) for striking highlights on call-to-action buttons.
- Headline font: 'Space Grotesk' for a technical feel; Body font: 'Inter' for legibility in long policy documents.
- Clean, line-based glyphs with subtle gradients mirroring the modern 'swiss army knife' nature of the toolbox.
- Bento-box style grid system to elegantly categorize the diverse feature sets into logical segments.
- Precision micro-interactions and smooth stagger-fades that simulate a speed-first technical environment.
